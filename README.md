How to Run My Website
1. Download the Project Files

Download the entire project folder, including:

* All HTML files
* All CSS files
* All images

Make sure everything is put in the same folder after downloading everything.

2. Open the Project in VS Code

1. Open Visual Studio Code.
2. Go to File > Open Folder.
3. Select the folder you put everything you downloaded inside of it.
4. Wait for VS Code to load all your files.

3. Run the Homepage

1. Inside VS Code, click on the file named homepage.html.
2. At the top, click Run Without Debugging.

After the Quiz

You receive your result.
Click the result to continue.

This takes you to the main feature:

Dating a Sentient Climate Disaster

This is the core of my project.

How It Works

* You read the dialogue
* Press Next to continue
* You will make at total of 3 choices
* Each choice path leads to:

a good ending, or a bad ending

If you get a bad ending, you can try again.

If you get the good ending, you will be sent back to the homepage, where you may restart the cycle.
